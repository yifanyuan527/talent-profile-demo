人才画像库 Demo 本地部署说明

包含文件：
- index.html：主页面，包含画像库和画像录入
- styles.css：页面样式
- app.js：画像评分、筛选、雷达图和问卷回写逻辑
- questionnaire.html：员工自评问卷页面

推荐打开方式：
1. 解压压缩包。
2. 双击 index.html，或在浏览器中打开 index.html。
3. 进入“画像录入”模块，点击“打开员工自评问卷”。
4. 问卷提交后，回到主页面，可看到“已收到”状态，再点击“生成画像”。

如果希望用本地服务打开：
1. 在该目录打开终端。
2. 执行：
   python3 -m http.server 5174
3. 在浏览器访问：
   http://127.0.0.1:5174/index.html

注意：
- 这是纯前端离线 Demo，不需要安装依赖。
- 当前不会真正解析 PDF，PDF 上传用于展示入口；摘要补充会参与 Demo 规则评分。
- 问卷结果保存在浏览器 localStorage 中，换浏览器或清缓存后会丢失。
